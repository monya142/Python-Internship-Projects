# Task 1: BMI Calculator

This project is a simple Python command-line application that calculates the Body Mass Index (BMI) of a user based on their weight and height.

## Objective
To calculate BMI and display the health category such as Underweight, Normal weight, Overweight, or Obese.

- Python
